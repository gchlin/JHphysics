波動物理大亂鬥 (Wave Physics Battle)這是一款專為物理教學設計的雙人互動網頁遊戲，專注於「駐波」、「泛音」與「諧音」的觀念辨析。透過手機或平板的分割畫面，讓兩位學生面對面進行搶答或競速對戰。🚀 特色功能雙人同屏對戰：介面自動旋轉 180 度，適合面對面遊玩。六大關卡設計：Level 1~3 (看圖選字)：泛音特訓、諧音特訓、混合題型。Level 4~6 (看字選圖)：逆向工程特訓。嚴格物理邏輯：系統強制隔離「泛音」與「諧音」選項，避免混淆（例如：正確答案是第一泛音，干擾項也全都是泛音）。清楚區分「兩端固定 ($n$)」與「一端固定 ($m$)」的物理特性。多種遊戲模式：練習模式：單人刷題，可隨時查看對照表。雙人搶答：先答對得 2 分，後答對得 1 分，搶 20 分獲勝。30秒競速：在有限時間內比拼正確率。📂 檔案結構說明本專案採用純前端架構 (HTML/CSS/JS)，無需後端伺服器即可執行。PlaintextMathGame/ (專案根目錄)

🛠️ 如何使用啟動遊戲：直接點擊兩下 index.html 即可在瀏覽器中開啟。若要在手機遊玩，請將整個資料夾傳輸至手機，或上傳至 Google Drive / 靜態網頁空間。修改題目 (data.js)：所有題目與選項都定義在 AnswerBank 與 QuestionList 中。若要新增圖片，請將圖片放入 assets/images/，並在 data.js 中新增對應條目。🎨 圖片命名規則 (物理定義)為了方便維護，圖片檔名採用以下縮寫：ff_n.png (Fixed-Fixed, 兩端固定):$n=1, 2, 3, 4...$ (波腹數量 = $n$)fo_n.png (Fixed-Open, 一端固定):檔名中的數字代表序號，但物理意義對應 奇數諧音。fo_1 $\rightarrow$ $m=1$ (基音)fo_2 $\rightarrow$ $m=3$ (第一泛音 / 第三諧音)fo_3 $\rightarrow$ $m=5$ (第二泛音 / 第五諧音)fo_4 $\rightarrow$ $m=7$ (第三泛音 / 第七諧音)⚠️ 注意事項安全性：本程式採靜態網頁設計，不含後端資料庫，無 SQL Injection 風險。題目渲染採用 textContent 與嚴格類型檢查，防止 XSS 攻擊。瀏覽器支援：支援所有現代瀏覽器 (Chrome, Safari, Edge, Firefox)。請確保瀏覽器未靜音以聽到音效。👨‍🏫 作者資訊設計製作：[您的名字] 老師技術協作：AI 輔助開發版本：1.0 (Physics Edition)
