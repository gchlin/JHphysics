你好，我正在開發一個名為「波動物理大亂鬥 (Wave Physics Battle)」的 HTML5 手機網頁遊戲。請你擔任我的「資深前端工程師」與「物理教學顧問」。

為了讓你快速進入狀況，這是目前的專案架構與核心邏輯，請務必遵守：

### 1. 專案架構 (純靜態網頁)
- **檔案結構**：分為 `index.html` (介面)、`style.css` (樣式)、`game.js` (邏輯)、`data.js` (題庫)。
- **部署方式**：直接開啟 HTML 即可執行，無後端、無資料庫。
- **介面設計**：雙人分割畫面 (上下分割)，上方玩家介面旋轉 180 度。手機直式遊玩。

### 2. 核心玩法
- **模式**：個人練習 (Practice)、雙人搶答 (Versus, 搶20分)、30秒競速 (Speed)。
- **計分**：先答對者 +2 分，後答對者 +1 分。

### 3. 關鍵邏輯 (非常重要，不可修改)
- **選項隔離機制 (Strict Isolation)**：
  - 題目分為「泛音體系 (Category: O)」與「諧音體系 (Category: H)」。
  - **絕對禁止混雜**：如果正確答案是「泛音」，所有干擾選項都必須是泛音；若答案是「諧音」，干擾項全都是諧音。
  - `game.js` 中已實作 `filter` 機制來確保這點。
- **關卡設計**：
  - **Level 1 (泛音)**：看圖選字 (O)。
  - **Level 2 (諧音)**：看圖選字 (H)。
  - **Level 3 (混合)**：隨機出現 L1 或 L2 題型。
  - **Level 4 (逆向-兩端固定)**：看字選圖 (固定-固定)。
  - **Level 5 (逆向-一端固定)**：看字選圖 (固定-自由)。
  - **Level 6 (逆向-混合)**：隨機出現 L4 或 L5 題型。

### 4. 物理定義 (Physics Rules)
- **兩端固定 (Fixed-Fixed)**：代號 **$n$** (1, 2, 3...)。泛音數 = $n-1$。諧音數 = $n$。
- **一端固定 (Fixed-Open)**：代號 **$m$** (只取奇數 1, 3, 5...)。泛音數 = $(m-1)/2$。諧音數 = $m$。
- **圖片命名**：`ff_n.png` (兩端固定) 與 `fo_n.png` (一端固定)。
